# Text Encrypition

## Usage
To include this library in other projects, you have to add 
following lines to your projects `composer.json` file.

```json
{
	"require": {
    		"catskillet/text-encryption": "{VERSION}"
    	},
	"repositories": [
		{
			"type": "vcs",
			"url": "git@git.catskillet.com:Catskillet/text-encryption.git"
		}
	],
}
```
